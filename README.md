# ⚗️ Prompt A/B Tester

> 두 개의 시스템 프롬프트를 동시에 테스트하고 AI가 점수를 평가해주는 프롬프트 엔지니어링 도구

**🔗 Demo:** https://westkong.github.io/prompt-tester/

---

## 📌 프로젝트 소개

프롬프트 엔지니어링을 공부하면서 느낀 불편함에서 출발했습니다.  
"이 프롬프트가 더 나은가, 저 프롬프트가 더 나은가?"를 직접 비교하고 싶었지만  
매번 따로 테스트하고 직접 비교해야 했습니다.  
같은 질문에 두 프롬프트를 동시에 실행하고, AI가 객관적으로 점수까지 매겨주는 툴을 만들었습니다.

---

## ✨ 주요 기능

| 기능 | 설명 |
|------|------|
| ⚔️ A/B 동시 실행 | 같은 질문에 두 프롬프트를 병렬로 실행 |
| 📊 3항목 점수 평가 | 정확성 / 명확성 / 유용성 각 10점 만점 |
| 🏆 승자 자동 판정 | AI가 종합 점수 기반으로 승자 결정 |
| 💬 AI 총평 | 두 프롬프트의 차이점과 개선 방향 제시 |
| ⚡ 병렬 API 호출 | A/B 동시 호출로 빠른 결과 |

---

## 🛠 기술 스택

- **Frontend:** HTML, CSS, Vanilla JavaScript (단일 파일)
- **AI:** Anthropic Claude API (`claude-sonnet-4-20250514`)
- **배포:** GitHub Pages

---

## 🚀 사용 방법

1. [사이트](https://westkong.github.io/prompt-tester/) 접속
2. Claude API 키 입력
3. 공통 질문 입력
4. 프롬프트 A / B 각각 입력
5. **RUN TEST** 클릭 → 결과 + 점수 확인

---

## 💡 활용 예시

**질문**: "블록체인이 뭔지 설명해줘"

| | 프롬프트 A | 프롬프트 B |
|---|---|---|
| 내용 | `당신은 친절한 AI입니다.` | `당신은 10년 경력 블록체인 전문가입니다. 비유를 활용해 3단계로 설명하세요.` |
| 정확성 | 7 | 9 |
| 명확성 | 6 | 9 |
| 유용성 | 6 | 9 |
| **합계** | **19** | **27** |

→ 역할 부여 + 구조화 지시만으로 점수가 크게 달라짐
